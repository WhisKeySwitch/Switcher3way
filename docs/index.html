Typed the right word in the wrong layout?

ghbdtn → привет

Switcher3way renders your keystrokes through every installed layout, validates each candidate against its own language's dictionary, and switches only when there's a single unambiguous winner — automatically as you type, or on a key tap.

Why it's different

N-way, not two-way

Most switchers flip between two layouts. Switcher3way considers all your installed layouts at once and picks the right one.

Precision-first

Words valid in more than one language (like там in Ukrainian & Russian) are left untouched — no wrong "corrections".

Auto or manual

Fix words automatically as you type, or tap a trigger key to convert the last word or selection — tap again to undo.

Stays out of your way

Per-app exceptions, never/always-convert word lists, password-manager safety, and pause (30 min / 1 h / until restart).

Little touches

Menu-bar status header, per-app layout memory, an optional layout flag at the cursor, and a UI localized in 16 languages.

Private & offline

Everything runs locally on your device. No accounts, no telemetry, no network.

Install

macOS

  1. Download the DMG, open it, and drag Switcher3way.app into Applications.
  2. First launch: the app is unnotarized, so right-click → Open once (double-clicking is blocked by Gatekeeper). macOS remembers the choice.
  3. Grant two permissions when asked — Accessibility (retype the fixed word) and Input Monitoring (see keystrokes). The onboarding checklist detects grants live.

Windows · preview

  1. Download the MSI and double-click it; approve the User Account Control prompt.
  2. This preview isn't code-signed yet, so SmartScreen may warn — click More info → Run anyway.
  3. Launch Switcher3way from the Start menu; it lives in the system tray. No permissions dialog and no .NET install needed — the runtime is bundled.
Windows is a 64-bit preview (unsigned; no auto-update yet). Prefer to build it yourself? git clone the repo — macOS: bash build_app.sh; Windows: pwsh windows/build-msi.ps1. See the README.
