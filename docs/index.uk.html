Switcher3way

Застосунок для macOS і Windows, який помічає мову, якою ви друкуєте, і виправляє розкладку клавіатури — для англійської, української та російської.

Набрали правильне слово в неправильній розкладці?

ghbdtn → привет

Switcher3way пропускає ваші натискання клавіш через кожну встановлену розкладку, перевіряє кожен варіант за словником його мови й перемикає лише тоді, коли є єдиний однозначний переможець — автоматично під час набору або за натисканням клавіші.

Чим це відрізняється

N-way, а не дві розкладки

Більшість перемикачів чергують лише дві розкладки. Switcher3way враховує всі встановлені розкладки одразу й обирає правильну.

Спершу точність

Слова, дійсні в кількох мовах (як-от там в українській і російській), залишаються незмінними — жодних хибних «виправлень».

Автоматично або вручну

Виправляйте слова автоматично під час набору або натисніть клавішу-тригер, щоб перетворити останнє слово чи виділення — натисніть ще раз, щоб скасувати.

Не заважає

Винятки для окремих застосунків, списки слів «ніколи / завжди перетворювати», захист менеджерів паролів і пауза (30 хв / 1 год / до перезапуску).

Приємні дрібниці

Рядок стану в меню, запам'ятовування розкладки для кожного застосунку, необов'язковий прапор розкладки біля курсора та інтерфейс, локалізований 16 мовами.

Приватно та офлайн

Усе працює локально на вашому пристрої. Жодних облікових записів, телеметрії чи мережі.

Встановлення

macOS

  1. Завантажте DMG, відкрийте його та перетягніть Switcher3way.app до Програм (Applications).
  2. Перший запуск: застосунок не нотаризований, тож один раз клацніть правою кнопкою → Відкрити (подвійний клік блокує Gatekeeper). macOS запам'ятає вибір.
  3. Надайте два дозволи, коли з'явиться запит, — Універсальний доступ (Accessibility, щоб перенабрати виправлене слово) та Моніторинг введення (Input Monitoring, щоб бачити натискання). Контрольний список під час налаштування виявляє надані дозволи наживо.

Windows · попередня версія

  1. Завантажте MSI і двічі клацніть його; підтвердіть запит Контролю облікових записів (UAC).
  2. Ця попередня версія ще не підписана, тож SmartScreen може попередити — натисніть Докладніше → Виконати в будь-якому разі.
  3. Запустіть Switcher3way з меню «Пуск»; він живе в системному лотку. Жодних діалогів дозволів і не потрібно встановлювати .NET — середовище виконання вбудоване.
Windows — це 64-бітна попередня версія (без підпису; поки без автооновлення). Волієте зібрати самостійно? git clone репозиторій — macOS: bash build_app.sh; Windows: pwsh windows/build-msi.ps1. Див. README.
