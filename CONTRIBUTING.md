# Contributing

Thanks for your interest in **Switcher3way**.

## This project is not accepting external contributions

Switcher3way is a personal project maintained by its author. To keep it focused:

- **Pull requests from outside the maintainer will not be reviewed or merged.**
- Feature requests are generally out of scope.

This isn't a reflection on you or your idea — it's just how this repository is run.
Please don't invest time in a PR expecting it to be merged.

## You're welcome to fork

The project is open source under the [MIT License](LICENSE) (itself a fork of
[rashn/RuSwitcher](https://github.com/rashn/RuSwitcher)). You are free to **fork it and
adapt it for your own use** within the terms of that license. A fork is your own copy —
build on it however you like.

## Bug reports

If Issues are enabled you may file one, but please note that responses and fixes are at
the maintainer's discretion and not guaranteed.

## Security

Please do **not** report security issues in a public issue. Instead, use GitHub's private
vulnerability reporting (the repository's **Security → Report a vulnerability** tab) so the
problem can be addressed before it's disclosed.
